# 🌍 Wanderlust — Travel Listing Platform

Wanderlust is a full-stack travel listing web application inspired by Airbnb, where users can explore, create, and review unique travel destinations across the world.

🚀 Built with scalability, clean architecture, and real-world backend concepts.

---

## 🚀 Live Demo

🔗 [Visit Wanderlust](https://wanderlust-fk9k.onrender.com)

---

## 📸 Features

### 🏡 Listings

* Browse beautiful travel destinations
* Clean card-based UI with pricing (₹/night)
* Category filters (Trending, Rooms, Cities, Mountains, etc.)
* Responsive grid layout

### 🔐 Authentication

* User Signup & Login
* Secure password handling
* Session-based authentication

### ✍️ CRUD Functionality

* Create new listings
* Edit existing listings
* Delete listings
* Ownership-based authorization

### ⭐ Reviews System

* Add reviews to listings
* User-generated feedback
* Linked review-listing structure

### ⚡ UX Enhancements

* Flash messages for actions
* Error handling middleware
* Clean navigation & layout
